# Security policy

## Supported versions

NimbleJS is pre-1.0. Security fixes target the latest commit and latest
published pre-release. Persistence adapters handle application-provided data;
NimbleJS does not encrypt, authenticate, or validate persisted values.

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting for
`onuracar-dev/NimbleJs`. Do not publish exploit details in an issue. Include the
affected version, reproduction, impact, and suggested mitigation when possible.
You should receive an acknowledgement within seven days.
